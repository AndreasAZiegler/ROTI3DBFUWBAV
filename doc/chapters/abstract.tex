\chapter*{Abstract}
Object tracking is an important part for many applications especially for robotic systems interacting with humans.

Ultra-wideband (UWB) systems as well as vision based object tracker are widely known and used. Both of the systems have their advantage and disadvantages. UWB systems can provide the location of an object in 3D with an accuracy of approximately $10\textit{cm}$ whereas vision based object tracker can only provide the location of an object in 2D pixel coordinates but with a more precise accuracy than UWB systems.

So why not combine this two sources of information? Exactly this concept should be developed and evaluated in this semester project. The 3D position measured by the UWB system should be fused with the 2D pixel coordinates of a visual based object tracker with and Extended Kalman Filter (EKF).

The proposed system shows a significantly better accuracy compared to the 3D position measured by the UWB system. This prove of concept enables to apply this systems to a wide range of applications and also for further extension.
